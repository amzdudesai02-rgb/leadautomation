# Frontend Application

## Setup

1. Install dependencies:
```bash
npm install
```

2. Create `.env` file:
```
VITE_API_URL=http://localhost:5000
```

3. Run development server:
```bash
npm run dev
```

## Build

```bash
npm run build
```

## Tech Stack

- React 18
- Material-UI
- Vite
- Axios
- React Router
